# HavenMC ToolBox - Official Extension

![HavenMC Logo](./HavenMC-ToolBox/havenmc.png)

HavenMC ToolBoxは、Minecraftサーバー「HavenMC」のプレイヤーのために開発された、Microsoft Edge公式のブラウザ拡張機能です。
ゲームにログインしていなくても、ブラウザから瞬時にサーバーの状況を確認し、コミュニティと繋がることができます。

## 主な機能

- **リアルタイム・プレイヤー統計:** 現在のオンライン人数、接続中のプレイヤー名（MCID）を一覧表示します。
- **ライブ・スキンビューアー:** 接続中のプレイヤーが現在使用しているスキンをリアルタイムでプレビューできます。
- **現在地の確認:** プレイヤーがサーバー内のどのワールド、どの座標にいるかを把握できます。
- **最新ニュースへのアクセス:** HavenMC公式YouTubeチャンネルの最新投稿や、公式X (旧Twitter) の最新ポストを即座にチェックできます。
- **公式リンク集:** ストア、公式Discord、投票サイトなど、HavenMC関連の重要リンクへワンクリックでアクセス可能です。

## インストール方法

本拡張機能は **Microsoft Edge Add-ons ストア** からインストールしてください。

## プライバシーとセキュリティ

HavenMC ToolBoxは、ユーザーのプライバシーを尊重します。

- **データ収集:** 本拡張機能はユーザーの個人情報を収集しません。表示されるプレイヤー情報はすべて公開されているサーバーデータに基づきます。
- **ローカルストレージ:** 設定の保存にはブラウザの `localStorage` を使用し、データはユーザーのデバイス内にのみ保持されます。

詳細は [PRIVACY.md](./PRIVACY.md) をご確認ください。
